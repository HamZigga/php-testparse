## Запуск проекта на docker

`cd docker`

`docker-compose build`

`docker-compose up -d`

`docker exec -i testparser-mysql-1 mysql -uroot -proot testparserDB < lotsinfo.sql`

внести домен bikeshop.test в hosts 127.0.0.1 bikeshop.test
